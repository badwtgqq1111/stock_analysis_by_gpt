| | print_section() | 格式化输出标题 |

| `main.py` | - | 编排 5 阶段管道 |

## 技术规格

### 数据来源
- **API**: 腾讯财经 (腾讯行情)
- **股票代码**: 03633 (中裕能源)
- **数据量**: 1000 条 K 线记录
- **时间范围**: 2022-02-11 至 2026-03-05
- **更新频率**: 实时

### 依赖包
```
requests==2.32.5      (HTTP 请求)
pandas==3.0.1         (数据处理)
matplotlib==3.10.8    (图表绘制)
```

### 输出格式
- **数据**: JSON 格式，包含元数据和完整 OHLCV 数据
- **图表**: 高质量 PNG (100 DPI)，自动布局适配

## 执行流程

主程序执行以下 5 阶段：

```
[INIT] 初始化
  └─ 清理历史数据
  └─ 创建输出目录

[MODULE] 阶段 1: 数据下载
  ├─ StockInfoFetcher.fetch() → 基本信息
  └─ HistoryDataFetcher.fetch() → 1000 条记录

[MODULE] 阶段 2: 数据显示
  ├─ 股票名称、价格、成交量
  ├─ 数据预览（最近 5 行）
  ├─ 统计信息（最高/最低/平均价格）
  └─ 列说明（5 列数据）

[MODULE] 阶段 3: 数据保存
  └─ DataSaver.save_json() → hk_stock_*.json

[MODULE] 阶段 4: 图表绘制
  └─ KLineChartPlotter.plot() → kline_chart_*.png

[DONE] 完成
  └─ 输出摘要信息
```
